# Hafiz Azrab — Portfolio

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Stack
- React 18
- React Router v6
- Tailwind CSS v3
- Framer Motion v11
- Lucide React (icons)

## Local Development

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to a GitHub repository
2. Import the repository at [vercel.com](https://vercel.com)
3. Vercel auto-detects Create React App — no config needed
4. The included `vercel.json` handles client-side routing

## Pages
- `/` — Home (hero, featured projects, services, CTA)
- `/about` — About (bio, values, education)
- `/projects` — Projects (filterable grid with search)
- `/skills` — Skills (animated bars + tool ecosystem)
- `/experience` — Experience (timeline + milestones)
- `/contact` — Contact (form + contact info)

## Customization
- Edit `src/data/portfolio.js` to update all personal info, projects, and experience
- Swap the avatar initials with a real image in `Home.jsx` and `About.jsx`
- Wire up the contact form to your backend, Formspree, or EmailJS
